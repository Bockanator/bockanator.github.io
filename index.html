Hi this is my website for the game Catch of the Day. It is recommended you click on the tutorial in game
before playing. There are some intricacies that are not taught in the short tutorial that you are intended to
learn playing the game. Sound will not work for the main menu until you have clicked onto the game once it has loaded.
